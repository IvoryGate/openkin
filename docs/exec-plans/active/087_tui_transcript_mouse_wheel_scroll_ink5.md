# 087 — 转录区：鼠标滚轮 / 「类网页」滚动

## 元信息

- **状态**: **调研 / 未实现**（父 [083](./083_opencode_zen_layout_parent.md)）  
- **问题**: Ink 5 `useInput` 的 `Key` 类型**不含**滚轮/鼠标；需 **stdin 原始流** 解析 SGR 鼠标序列（如 `\x1b[M...`）或依赖终端的 **交替屏幕 + 1000/1002/1003** 模式。  
- **可能路径**  
  1. **主路径**: `useStdin` raw + 有限状态机解析，与现有 `useInput` 协调（避免双读）。  
  2. **替代**: 强化 **Up/Down**、**PgUp/PgDn** 的「网页式」滚动与提示文案，不依赖鼠标。  
- **风险**: tmux/SSH/不同终端对鼠标报告支持不一；与 Ink 内部 stdin 读冲突。  
- **建议**: 先完成 **084–086**；087 单独立项验证环境后再编码。

## 验收（若实现主路径）

- 在 iTerm/Terminal.app + 本机 TTY 上滚轮能上下滚 transcript，且不吞输入。
