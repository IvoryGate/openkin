# 153-B4 — 群成员管理面板（添加/移除/解散）

> **状态**：待执行
> **前置**：153-B3
> **预估**：1d

---

## 目标

右栏信息面板展示群成员列表，支持添加/移除成员、修改群名、解散群聊。

---

## 核心设计

### 群信息面板

- 群名编辑（双击可修改，调用 `PATCH /v1/sessions/:id` 更新 displayName）
- 群成员列表：
  - 每个 Agent 显示头像 + 名称 + 在线状态
  - 点击 Agent 查看详情
  - "移除"按钮（确认后移除，更新 `conv.agentIds`）
- "添加成员"按钮 → 弹出 Agent 选择 Flyout
- "解散群聊"按钮 → 确认后删除

### 群成员变更

- 添加成员：更新 `conv.agentIds`，不需要新建 Session
- 移除成员：同上
- 解散群聊：从 `channelConversations` 中移除，可选删除后端 Session

---

## 验收标准

1. ✅ 右栏展示群成员列表
2. ✅ 可添加新 Agent 到群聊
3. ✅ 可移除群内 Agent
4. ✅ 可修改群名
5. ✅ 可解散群聊
6. ✅ 成员变更后消息路由自动更新
