# 153-B4 — 群成员管理面板（添加/移除/改名/解散）

> **状态**：✅ 已实现
> **前置**：153-B1（已完成）
> **预估**：1d

---

## 目标

在群聊信息面板中提供完整的群管理能力：改名、添加/移除成员、解散群聊。

---

## 已实现功能

### 群聊设置面板

选中群聊后，右栏信息面板显示：
- 群组合头像 + 群名 + 成员数
- "群聊设置"按钮（点击展开/收起）
- 群成员列表

### 设置面板功能

1. **改名**：输入新群名 → 点击保存 → 即时生效
2. **添加成员**：列出不在群中的 Agent → 点击即添加
3. **移除成员**：每个 Agent 旁有 ✕ 按钮 → 点击即移除
4. **解散群聊**：红色按钮 → confirm 确认 → 删除群组数据和消息 → 回到空状态

### 交互细节

- 用户（自己）显示"群主"标签，不可移除
- 添加/移除后自动刷新信息面板和联系人列表
- 解散后重置聊天区域为空状态
- 所有操作有 Toast 通知反馈
