# 116 · Desktop 中下固定输入区与工具栏增强

## 任务边界

本单在 `115` 基础上，为桌面端中间输入区补齐完整交互壳层：

1. 输入区固定在中间区域下部，不随消息区滚动离开视口
2. 输入区底栏提供以下可交互入口：
   - 模型选择
   - 联网开关
   - 上传附件
   - 图片上传
   - 允许完全控制电脑开关
   - 上下文入口
3. 右侧栏新增上下文展示区，与输入区「上下文」入口联动
4. 输入区入口统一采用 icon + 文案的桌面设计（不使用 emoji）

## 影响范围

- `apps/desktop/renderer/index.html`
- `apps/desktop/renderer/styles.css`
- `apps/desktop/renderer/app.js`

## 不做什么

- 不扩展后端 API / contract
- 不引入新的第三方状态管理或 UI 框架
- 不在本单实现真实附件上传协议，仅做入口壳层与交互状态

## 验收标准

- 输入区固定在中下部，窗口尺寸变化时位置保持稳定
- 工具栏含模型、联网、附件、图片、完全控制电脑、上下文入口
- 工具栏按钮有可见状态反馈（选中/未选中）
- 右侧栏可展示上下文概览，并可被「上下文」入口触发聚焦
- 保持现有会话/消息主流程可用（发送消息与刷新逻辑不回退）
- `pnpm --filter @theworld/desktop check` 通过
