# 迭代 3：UI 界面重新设计

## 1. 迭代概述

### 1.1 迭代目标
本迭代的目标是参考 DESIGN.md 设计规范，重新设计整个 UI 界面和风格，提升用户体验和视觉美感，同时保持现有功能不变，仅改变布局和风格。

### 1.2 核心价值
- **提升视觉体验**：采用"数字书斋"设计语言，营造宁静、专注的数字环境
- **改善阅读体验**：纸质书布局风格，增加垂直留白，优化排版
- **统一设计语言**：应用一致的颜色系统、字体和组件样式
- **保持功能完整**：不修改任何现有功能，仅优化界面呈现

## 2. 设计理念

### 2.1 设计主题：Digital Study（数字书斋）
本次 UI 重新设计的核心是"数字书斋"概念，将物理书斋的体验转化为数字界面：
- **材质感**：模拟和纸的质感，使用暖色调和柔和阴影
- **层次感**：通过色调变化而非硬边框来创建层次
- **呼吸感**：增加垂直留白，减少视觉压迫
- **专注感**：简洁的界面，减少干扰元素

### 2.2 核心设计原则
1. **色调分层**：使用色相变化而非边框来区分区域
2. **柔和阴影**：使用"幽灵阴影"代替硬朗阴影
3. **不对称布局**：打破模板化布局，增加自然感
4. **阅读节奏**：宽边距和充足的垂直留白

## 3. 主要变更

### 3.1 颜色系统
从原来的深色主题改为自然土色调的浅色主题：

| 角色 | 原颜色 | 新颜色 | 说明 |
|-----|--------|--------|------|
| 主背景 | `#0F172A` | `#FBF9F4` | 未漂白纸色 |
| 主色调 | `#3B82F6`（蓝） | `#5E5E5E`（深炭） | 高对比度文本 |
| 次要表面 | `#1E293B` | `#F5F4ED` | 侧边栏 |
| 高亮表面 | `#334155` | `#E8E9E0` | 激活状态 |
| 纯白 | - | `#FFFFFF` | 交互卡片 |

### 3.2 字体系统
引入新的字体组合：
- **衬线字体（Noto Serif）**：用于标题和展示元素，传递学者气质
- **无衬线字体（Manrope）**：用于正文和 UI 元素，保证可读性

### 3.3 布局变更

#### 3.3.1 ChatPage：三栏布局
- **左栏**：导航和 Agent 列表（Layer 1）
- **中栏**：主要对话区域（Layer 0）
- **右栏**：工具和上下文面板（可折叠，Layer 1）

#### 3.3.2 顶部栏：毛玻璃效果
- 使用 `backdrop-filter: blur(20px)`
- 背景透明度 80%
- 内容可透过，营造层次感

#### 3.3.3 消息列表：书籍布局
- 增加垂直留白（`py-16`）
- 最大宽度限制（`max-w-3xl`）
- 消息内容使用纯排版，无背景色块

#### 3.3.4 输入栏：浮动胶囊
- 圆角 `xl`（1.5rem）
- 绝对定位在底部
- 幽灵阴影效果

### 3.4 组件样式更新

#### 3.4.1 按钮
- **主按钮**：深炭色背景，浅色文本，柔和阴影
- **次要按钮**：背景色变化，无边框
- **幽灵按钮**：无背景，hover 时表面变化

#### 3.4.2 输入框
- 使用"幽灵边框"：15% 不透明度的边框
- focus 状态使用柔和的阴影环
- placeholder 使用次要色调

#### 3.4.3 卡片
- 白色背景，柔和阴影
- hover 时阴影加深，背景色变化
- 无边框设计

#### 3.4.4 头像
- 自然色调：棕色、橄榄绿、蓝灰色、赭色等
- 圆形设计
- 较大尺寸（40-44px）

### 3.5 交互体验优化
- **过渡动画**：统一使用 `duration-300`，更加平滑
- **Hover 反馈**：表面颜色变化，而非边框变化
- **焦点状态**：使用柔和的阴影环，而非硬边框
